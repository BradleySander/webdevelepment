demo 2 session storage

-Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent?
Neen het is weg na het sluiten van de pagina.

-Blijft de data behouden als je de pagina herlaadt?
Ja na het herladen van de pagina blijft de score behouden.

-Wordt de data gedeeld door verschillende tabbladen als je de pagina meermaals opent?
Neen want dit is een session een andere tabblad is een nieuwe session.

-Denk je dat een andere pagina (andere file of url) aan de data geraakt als je ze in hetzelfde
tabblad opent? Neen want dit is een session storage deze blijft in het domein.

demo 3 local storage

-Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent?
Ja, local storage behoudt data zelfs na het sluiten en opnieuw openen van het tabblad.

-Blijft de data behouden als je de pagina herlaadt?
Ook bij het herladen van de pagina blijft de data behouden.

-Blijft de data behouden als je de browser afsluit, opnieuw start en de pagina weer opent?
Zelfs na het afsluiten en herstarten van de browser blijft de data bewaard.

-Blijft de data behouden als je de pagina in een andere browser opent?
Data wordt niet gedeeld tussen verschillende browsers.

-Wordt de data gedeeld door verschillende tabbladen als je de pagina meermaals opent?
Ook tussen tabbladen van dezelfde browser wordt de data niet gedeeld.

-Denk je dat een andere pagina (andere file of url) aan de data geraakt?
Een andere pagina kan niet aan de local storage data komen omdat het per domein is.
