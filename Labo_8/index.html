Opdracht 3

Waarom staat de tekst in de blokken horizontaal gecentreerd? text-align:center;

Ga na dat een property vertical-align: center; geen effect heeft. Waarom eigenlijk niett? het is bedoeld om de verticale uitlijning van inline elementen in het midden van de hoogte van de omliggende tekst of inline box te plaatsen, niet om tekst in blokken verticaal te centreren.

Ga na dat vertical-align: middle; Het er niet voor zorgt dat de tekst verticaal gecentreerd wordt.

Wat doet vertical-align dan wel? De vertical-align CSS eigenschap regelt de verticale uitlijning van inline en inline-block elementen ten opzichte van de omliggende inhoud.

Zoek op Google of er een simpele manier is om tekst verticaal te centreren in een element. Tekst verticaal te centreren in een element, vooral als het gaat om meerdere regels tekst, kan je de 'Flexbox' CSS-layouttechniek gebruiken bv

.container {
display: flex;
justify-content: center; /* Horizontaal centreren */
align-items: center; /* Verticaal centreren */ }

• Wat is de ‘simpele’ manier om een één enkele regel tekst verticaal te centreren? (tip: line-height) mijn voorbeeld:

.vertical-center {
height: 100px; /* Hoogte van het element */
line-height: 100px; /* Maakt de tekst verticaal gecentreerd */ }

• Welke truc wordt vaak gebruikt om meerdere regels verticaal te centreren? (tip: table cell) Een veelgebruikte methode om meerdere regels tekst verticaal te centreren is door het gebruik van CSS, specifiek met behulp van `display: table` en `display: table-cell` voor de container en de inhoud ervan. Dit wordt gecombineerd met `vertical-align: middle` om de tekst verticaal te centreren binnen de container.

In 'styles.css' staan twee verschillende CSS regels die de height instellen van een element met class="area tall". Waarom krijgt de regel voor class 'tall' voorrang? omdat deze specefieker is goe specifieker het is of een hogere specificiteitsniveau heeft, doorgaans voorrang krijgen boven een meer algemene regel.
