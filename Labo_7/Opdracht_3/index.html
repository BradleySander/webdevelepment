Vraag

De regel .onbekend {background-color:red;} heeft geen effect op een <td> cel met class "onbekend" binnen een <tr> rij met class "alt" vanwege specifieke stijlen die al zijn gedefinieerd voor "alt" rijen. Om dit op te lossen, moet een specifieke regel toegevoegd worden, zoals tr.alt td.onbekend { background-color: red; }, om de rode achtergrondkleur te laten gelden voor cellen met class "onbekend" binnen "alt" rijen.
